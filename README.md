## This Project Needed
<a href="https://nodejs.org/en/">Nodejs</a>


### Install
Clone this project

```
> git clone https://github.com/X-SyntaxError/DLSyntaxBot
> cd DLSyntaxBot
```

Install the script:

```
> npm install
```

### Usage
1. Run The Script

```
> npm start
```
Or
```
> node index.js
```


2. Stop The Script

```
> ctrl + c
```
